Single-cell analysis of cancer | Bio-IT hackathon 2019

Single-cell RNA-seq can be used to infer copy number variations. Below, four brain cancer samples show large deletions in chr1p and chr19q, among other cytogenetic abnormalities.

This prototype reproduces part of a study in Single Cell Portal using inferCNV workflow output as run on Terra. Visualizations use Ideogram.js.

This prototype reproduces part of a study in Single Cell Portal using the Google Cloud Storage API to fetch inferCNV workflow output as run on Terra. Visualizations use Ideogram.js.
